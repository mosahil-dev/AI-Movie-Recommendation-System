# 🎬 AI Movie Recommendation System

An AI-powered Movie Recommendation System built using **Python, Flask, and Machine Learning**. This application recommends similar movies based on the selected movie using a TF-IDF similarity model.

---

## ✨ Features

- 🎥 AI-based Movie Recommendations
- 🤖 Machine Learning Recommendation Engine
- ⭐ IMDb Ratings
- 🎬 Netflix & IMDb Links
- 🖼️ Movie Posters
- 📅 Release Date
- 📝 Movie Summary
- 💎 Modern Glassmorphism UI
- 📱 Fully Responsive Design

---

## 🛠️ Tech Stack

- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- HTML5
- CSS3

## 🚀 Installation

### 1. Clone the repository

```bash
git clone <repository-url>
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Generate similarity matrix

```bash
python build_similarity.py
```

### 4. Run the application

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
